# React-Demo
### 基于Reactjs && Node 通讯的简单演示

* Reactjs && ES6 && webpack构建移动端京东首页
* 基于文件系统的Node.js服务端

## 运行程序

分别进入根目录和server目录下安装依赖包

```
> $ npm install
```

在server目录下启动后台服务

```
> $ npm start
```

在根目录下运行， 打开浏览器 `http://localhost:3000/`

```
> $ npm start
```

发布项目文件命令

```
> $ npm run build
```
